### 📂 Опис проєкту
Додаток є інтерактивним Web-середовищем для вивчення іноземних слів, логіку якого реалізовано за допомогою **Vanilla JavaScript**. 

**Ключові архітектурні та функціональні рішення:**
* **MVC Патерн:** Головну (робочу) сторінку додатку реалізовано за архітектурним шаблоном MVC (Model-View-Controller) з чітким розділенням даних, інтерфейсу та бізнес-логіки.
* **Збереження даних:** Як сховище даних використовується `localStorage` браузера для збереження стану додатку між сесіями.
* **Авторизація:** Реалізовано функціонал реєстрації, входу та виходу з системи. Прогрес вивчення слів зберігається індивідуально для кожного користувача.
* **Режими роботи:**
  * *Режим вивчення:* перегляд карток зі словами, відмітка "Знаю!" або "Ще вчу" з автоматичним підрахунком прогресу.
  * *Режим контролю знань:* генерація випадкових слів для перевірки перекладу з валідацією введених даних.
* **Управління профілем:** Можливість переглядати та редагувати особисті дані (ім'я, дата народження, аватар) з миттєвим оновленням інтерфейсу.
